# fasttravelbe
POC ride sharing app


## Run Locally

Clone the project

```bash
  git clone https://github.com/erajaykumar/fasttravelbe.git
```

Go to the project directory

```bash
  cd .\fasttravelbe
```

Install dependencies

```bash
  npm install
```

Start the server in production mode

```bash
  npm run start
```

Start the server in development mode

```bash
  npm run dev
```
